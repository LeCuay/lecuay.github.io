Proyectos

Oil & Rope

Oil & Rope

Oil & Rope es un proyecto web que pretende ayudar a gestionar partidas de rol de manera sencilla e intuitiva, tanto para aquellos que son expertos como para los que stán dando sus primeros pasos.

El proyecto es ambisioso y pretende conectarse a la API de Discord para que las acciones realizadas en la web se reflejen en un servidor o grupo de Discord.
De esta manera los jugadores tendrán dos formas de jugar, teniendo cada una sus ventajas (en la web un tablero gráfico, en Discord música, etc).

El código fuente del proyecto se encuentra en GitHub y el proyecto en producción puede ser visitado a través del siguiente enlace.

Coraline Project

Coraline Project

Coraline es un proyecto de inteligencia artificial escrito en Python cuyo objetivo principal es ser un apoyo conversacional a aquellas personas con problemas a la hora de relacionarse socialmente.

Coraline comenzó como un experimento para aprender Machine Learning pero, al ver el potencial del mismo, decidí enfocarlo de otra manera.
Ya no iba a ser un chatbot que me ayudara en mis tareas, sino que podría ser de mucha ayuda a aquellas personas que por motivos personales o externos sufren de distanciamiento social o simplemente les cuesta relacionarse con otras personas.

El proyecto aún no está preparado para ser lanzado de manera abierta, pero mi objetivo es montar unos cimientos sólidos y entregarlo a la comunidad.
